# Extract Frames from Video or Folder

Generic function to extract frames from a video file or all videos in a
folder.

## Usage

``` r
extract(extractor, verbose = TRUE, ...)
```

## Arguments

- extractor:

  An object of class `VideoFrameExtractor` or `FolderExtractor`.

- verbose:

  logical. If TRUE, progress messages are printed.

- ...:

  Additional arguments.

## Value

The extractor object (invisibly).
